# OG-LANS

OG-LANS is a research codebase for document-level event extraction on DuEE-Fin.

## Entry Points

- `main.py`: training
- `evaluate.py`: local base/checkpoint evaluation
- `evaluate_api.py`: API baseline evaluation
- `scripts/run_mini_matrix.py`: mini end-to-end comparison matrix

## Core Documents

- `AGENTS.md`
- `DATASET_DUEE_FIN.md`
- `ENGINEERING_STATUS.md`
- `OG_LANS_METHODOLOGY.md`

This repository is organized as an experiment workspace rather than a general-purpose packaged product.
